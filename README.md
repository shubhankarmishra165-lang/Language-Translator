# 🌍 Language Translator

A simple AI-powered Language Translator built using Python, Gradio, and Gemini API.  
This application can translate text into multiple world languages using Google's Gemini model.

---

## 🚀 Features

- Translate text into 80+ languages
- Simple and clean Gradio UI
- Powered by Gemini 2.5 Flash
- Fast and accurate translations
- Supports Indian and international languages

## ⚙️ Installation

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/LanguageTranslator.git
```

### 2️⃣ Move into the Project Folder

```bash
cd LanguageTranslator
```

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 🔑 Setup Environment Variables

Create a `.env` file in the project folder and add your Gemini API key:

```env
GEMINI_API_KEY=your_api_key_here
```

---

## ▶️ Run the Project

```bash
python app.py
```
